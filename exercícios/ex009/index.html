Trabalhando com listas

Listas Ordenadas:

Listas onde a ordem dos itens é importante.
Para criar uma lista ordenada, usamos a tag <ol>

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Assistir as aulas
  5. Almoçar

Listas não ordenadas:

Listas onde a ordem dos itens não faz diferença.
Para criar uma lista não ordenada usamos a tag <ul>

  • Item A
  • Item B
  • Item C
  • Item D
  • Item E

Os pontos da lista podem ser personalizados como circle, square e disc.
